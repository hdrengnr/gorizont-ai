Gant Protocol

Gant Protocol - это открытый унифицированный протокол для структурированного описания проектов, задач и процессов с использованием стандартных форматов данных (YAML/JSON).

Ключевые принципы

Открытость

Спецификация протокола доступна всем. Любая система может читать и записывать данные в формате Gant.

Простота

Использование стандартных форматов (YAML/JSON) делает протокол понятным как для людей, так и для машин.

Расширяемость

Базовая структура поддерживает добавление кастомных полей без нарушения совместимости.

Версионность

Поддержка миграций между версиями протокола гарантирует долгосрочную совместимость.
