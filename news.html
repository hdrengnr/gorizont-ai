Этап Pre-Production: тестирование и подготовка к запуску

Гант АСУП переходит в стадию Pre-Production Beta. Мы завершили реализацию ключевых возможностей платформы и приступили к финальному тестированию перед коммерческим запуском.

Что уже работает:

  • Gant Protocol — управление проектами через YAML/JSON
  • Dashboard с метриками и аналитикой
  • Multi-tenancy для команд и организаций
  • Security features: Encryption at Rest (AES-256), Audit Log
  • Интеграции: GitHub webhooks, 1C OData, Email (IMAP)
  • WebAuthn/Passkeys аутентификация

Что тестируем сейчас:

  • Стабильность работы под нагрузкой
  • Корректность работы backup/restore процедур
  • Compliance с требованиями 152-ФЗ и GDPR
  • Документация API и user guides

Коммерческий запуск планируется в Q1 2026. Для получения раннего доступа: contact@gorizont.ai
