Legal/Operational менеджер

Полная занятость

Юридическое сопровождение IT-компании, операционное управление.

Обязанности:

  • Юридическое сопровождение IT-компании (договоры, лицензии)
  • Compliance с 152-ФЗ, GDPR
  • Договорная работа (клиенты, подрядчики, НДА)
  • Операционное управление (документооборот, процессы)

Требования:

  • Высшее юридическое образование
  • Опыт работы в IT-компаниях от 2 лет
  • Знание IT-законодательства РФ (152-ФЗ, IT-аккредитация)
  • Английский язык (B2+) для работы с международными договорами
  • Навыки операционного управления и автоматизации процессов

Условия:

  • Конкурентная зарплата
  • ДМС
  • Обучение за счёт компании

DevOps-инженер

Полная занятость

Поддержка инфраструктуры Гант АСУП, CI/CD pipeline, мониторинг и on-premise развёртывание для enterprise клиентов.

Обязанности:

  • Поддержка и развитие инфраструктуры (Docker, Kubernetes)
  • Настройка и оптимизация CI/CD pipeline (GitHub Actions)
  • Мониторинг и логирование (Prometheus, Grafana)
  • Backup & Recovery стратегии
  • On-premise развёртывание для enterprise клиентов
  • Security hardening (SSL/TLS, firewalls, audit)
  • Автоматизация операций через IaC (Terraform/Ansible)

Требования:

  • Опыт работы DevOps-инженером от 2 лет
  • Docker, Kubernetes (обязательно)
  • Linux administration (Ubuntu/Debian)
  • Terraform или Ansible — плюс
  • Python/Bash scripting
  • Опыт с cloud провайдерами (AWS/Yandex Cloud)
  • Понимание security best practices

Условия:

  • Конкурентная зарплата
  • Современный стек технологий

AI/ML-инженер

Полная занятость

Интеграция LLM в продукты компании, разработка AI-автоматизаций, research новых технологий в области AI.

Обязанности:

  • Интеграция LLM в продукты
  • Разработка AI-автоматизаций для управления проектами
  • Создание RAG систем для работы с документацией
  • Fine-tuning моделей под задачи компании
  • Prompt engineering и оптимизация производительности
  • Research новых AI-технологий и трендов
  • Построение evaluation pipeline для LLM

Требования:

  • Опыт работы с LLM
  • Python (PyTorch/TensorFlow/Transformers)
  • Знание RAG, embeddings, vector databases (Pinecone/Weaviate)
  • Опыт fine-tuning моделей — плюс
  • Английский язык (B2+)
  • Понимание MLOps практик

Условия:

  • Конкурентная зарплата
  • Доступ к передовым AI-технологиям
  • R&D культура, свобода экспериментов
